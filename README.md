## ✨ Overview

This is a modern, premium, and conversion-optimized website built for **Kanan Dental Multispeciality Hospital** located in Lucknow. The website is designed to be fully responsive (mobile-first), lightning-fast (Lighthouse score 95+), and optimized for local SEO.

## 🚀 Features

- **Premium UI/UX:** Built with a trustworthy Navy Blue & Cyan color palette (`#0A2540`, `#00B4D8`) and modern typography (`Outfit` & `Inter`).
- **High Performance:** Bundled and optimized using **Vite** for minimal payload and lightning-fast loading speeds.
- **Scroll Animations:** Smooth scroll-reveal animations using the native `IntersectionObserver` API.
- **Conversion-Optimized CTAs:** Sticky floating buttons for WhatsApp, Phone Calls, and Appointment Booking.
- **SEO Ready:** Includes `LocalBusiness` JSON-LD schema markup, Open Graph tags, and optimized meta descriptions.
- **Interactive Elements:** Custom mobile hamburger menu and an interactive FAQ accordion.

## 🛠️ Tech Stack

- **Structure:** Semantic HTML5
- **Styling:** Custom Vanilla CSS3 with CSS Variables (No heavy frameworks)
- **Logic:** Vanilla ES6+ JavaScript
- **Build Tool:** Vite (Template Vanilla)

## 💻 Running Locally

To run this project on your local machine for development:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ashish7802/Kanan-Dental-Multispeciality.git
   cd Kanan-Dental-Multispeciality
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
